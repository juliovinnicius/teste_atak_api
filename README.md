## Baixando o projeto
No seu terminal, cmd, power shell ou bash execute os seguintes comandos:
- `git clone https://github.com/juliovinnicius/teste_atak_api.git`
- `cd teste_atak_api`
- `npm i`

### Rodando projeto
> Instalar as dependências da aplicação
```bash
npm install
```

> Iniciar a aplicação
```bash
npm run dev
```
